# Retail Customer Segmentation Analysis (RFM) 📊

### 🎯 Project Overview
This project analyzes a retail dataset of **23,000+ transactions**. I used the **RFM (Recency, Frequency, Monetary)** model to understand customer behavior and group them into actionable segments.

### 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn
- **Environment:** Google Collab

### 🔍 Key Insights
- **Best Customers:** Identified the top 10% who contribute to maximum revenue.
- **At-Risk:** Found customers who haven't shopped in 6+ months.
- **Visuals:** Created Heatmaps and Bar charts to show segment distribution.
